Domains without enforcement (no DMARC or p=none) [See Readme for Criteria]

Even if your domain is not intended to send email it should still have DMARC of p=reject and be parked, for example, v=spf1 -all which permits no IP to send mail. Follow Guidance here: https://www.ncsc.gov.uk/blog-post/protecting-parked-domains

Note: Companies or Entites Listed here may have DMARC on other domains.

Also Note if you use Entra/Microsoft 365 you still need to DMARC your .onmicrosoft.com domain. https://o365info.com/dkim-dmarc-onmicrosoft-com-domain/. This list will not include .onmicrosoft.com domains.
